# Short Description of the Project

This project is a high-performance Real-Time AI Investment Intelligence & Analytics Dashboard built using React, TypeScript, and Tailwind CSS. Powered directly by the Google GenAI SDK (Gemini), it allows users to search any public stock, index, or mutual fund to instantly generate comprehensive financial reports.

# Core Accomplishments & Capabilities:

**Dynamic Valuation Ratios:** Compares the target company's current P/E (Price-to-Earnings) and P/B (Price-to-Book) ratios directly against live average Sector/Industry benchmarks to evaluate pricing and premium opportunities.

**Micro-Structured Volatility Metrics:** Computes precise historical daily returns from the trailing 30 pricing cycles to render indicators for Annualized Volatility (%), Average Daily Moves (±%), Day-to-Day Standard Deviation, and Max Drawdown.

**Futuristic AI Score Ring:** Features an interactive progress-indicator gauge styled with orbiting holographic sub-layers and custom ambient color gradients representing low (green), moderate (amber), or high-risk (rose) asset characteristics.

**Search Auto-Suggestions:** Displays real-time matching suggestions categorized by search-prefix letters while typing, with a dedicated watchlist persistence container.
